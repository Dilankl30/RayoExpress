# RayoExpress

Plataforma de delivery multirol basada en React, TypeScript, Vite y Supabase.

## Desarrollo

```bash
npm install
npm run dev
```

## Calidad

```bash
npm run typecheck
npm run lint
npm run test
npm run build
```

## Configuración

Usa `.env.example` como referencia para las variables de Supabase.
